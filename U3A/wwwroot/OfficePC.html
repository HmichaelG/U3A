Setup Your U3A Office PC (Windows Only)

Your members' privacy is important

It is strongly recommended that office computers used to access the U3Admin website be locked down to prevent passwords and credit card details being stored on the machine. Use the following tools to prepare your Office PC for use with U3Admin

Note: You must have administrative priviledge on the PC to perform these tasks.

Install the U3A Desktop application

The U3A Desktop application provides a secure environment in which to execute the U3Admin software. It is a standard Windows application providing an icon on the desktop and an entry on the Windows Start menu. It can be added to the Windows taskbar if required.

Click here to download U3A Desktop

Lockdown your Internet browsers

Click the button below to downlaod registry settings that force your Internet browsers (Microsoft Edge and Google Chrome) to use in-private / incognito mode. Again, this will prevent login detail and credit card numbers from being saved to the machine.

Once downloaded click on the file to install it. Note that on download your browser will warn that this is a potentiall harmful download. This is normal.
